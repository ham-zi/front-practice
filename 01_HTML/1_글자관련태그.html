문단을 나누는 태그

문단 영여을 나누는 태그로는 p태그와 pre가 있다. 태그의 경우 개행을 하더라도 줄바꿈이 일어나지 않음

이런식으로 p태그를 하나 더 작성함 

        pre의 경우 브라우저에 자체 스타일이 부여되어 있음
        개행도 하면 개행이 잘됨
        -> 장점: 자유롭게 쓸 수 있다
        -> 단점: 브라우저에서 스타일이 부여되어 있음 (안예쁨)
가끔쓰는 태그 텍스트를 적는데 덩그러니 이렇게 적지 않음

b : 글자를 굵게 bored(시각적 강조)

strong : 글자를 굵게 만듦(의미적 강조)

i:글자를 기울여서 표기 (시각적 기울임)

em : 글자를 기울여서 표기(의미적 강조)

s : 글자에 취소선을 표현할 때 사용

mark : 형광펜 효과
