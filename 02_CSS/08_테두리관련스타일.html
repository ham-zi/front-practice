테두리 관련 스타일 ~(border~) 

        선택자 {
            테두리 스타일
            border-위치-style : none / dotted /
            dashed / soild / double / groove /
             ridge / inset / outset
            이 중에서 많이 쓰이는건 solid 이다.

            테두리 두께
            border-위치-width : 테두리 두께 px;

            테두리 색상
            border-위치-color : 색상값;

            테두리의 모서리를 동그랗게
            border-위치=radius : 숫자 px, %;

            => 모든 위치는 생략 가능

            그림자 
            box-shadow : 가로(x) 세로(y) 흐림정도, 번짐정도, 색상
        }
