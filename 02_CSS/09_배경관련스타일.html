7분 있으면 쉬는 시간

배경으로 이미지 넣기 

        선택자 {
            background-image : url(이미지 경로);
            background-repeat : repeat / repeat-x / repeat-y / no-repeat
            background-size : auto / contain / over / 숫자(%)
            background-prosition : left / right / center;
                                    top / center / bottom;
            background-attackment : scroll / fixed;
        }
