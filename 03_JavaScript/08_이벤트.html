이벤트

고전(기본) 이벤트 모델

        해당 요소 객체의 이벤트 속성에 접근해서
        이벤트 핸들러를 대입하는 방식

        객체명.(이벤트)속성명 = () => {} 
        참고, null을 대입하면 이벤트가 제거됨

인라인 이벤트 모델


        요소 내부에 직접적으로 이벤트 속성을 기술해서 정의하는 방식
        하지만 너무 많이 달아버리면 내용이 길어지고 복잡하고 가독성이 떨어지기 때문에
        보통 js모듈에 정의한 함수를 호출하는 방법을 선호한다.

표준 이벤트 모델 (addEventListener) 

        요소.addEventListener('이벤트명', 이벤트핸들러);

현재 이벤트가 발생한 이벤트 타겟에 접근하는 방법

태그별 기본적 달려있는 이벤트 제거하기

        - a : 클릭 시 href속성값으로 url요청을 보내는 이벤트
        
        - form : 태그 안에 있는 submit버튼 :
        클릭 시 사용자의 입력값들을 action 속성 값으로 요청 보내는 이벤트
