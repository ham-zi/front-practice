정규표현식


        정규표현식(Regular Expression : REGEX / REGEXP)이란,

        특정 패턴을 가진 문자열을 찾거나 찾아서 변경할 때 사용하는 '형식 언어'
        
        자바 => String 클래스 => matches() 

        정규표현식을 사용하면 조건검사하거나 값을 변경할 때
        조건문 /반복문을 사용하지 않고 간단히 처리할 수 있음

        자바스크립트 뿐만아니라 자바/오라클에서도 사용할 수 있음

* 메타문자 

            메타문자를 사용해서 "검색하고자하는 패턴"을 생성할 수 있음

* 플래그 문자


            플래그문자는? /정규표현식/ 뒤에 사용한다.

            종류
            i : 대/소문자 전부 검사
            g : 문자열 내의 모든 패턴 검사(전역으로 보겠다)
            m : 여러 줄을 전부 검사(개행문자까지 포함)

정규표현식을 이용해서 주민등록번호 검사
