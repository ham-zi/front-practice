자바스크립트의 경우 자료형을 선언하지 않음

자바스크립트는 총 세개의 키워드로 변수를 선언할 수 있음

var, let , const

변수 선언 위치에 따른 전역 / 지역 변수로 나뉜다

자료형 


            변수 선언 시 자료형을 별로도 선언하지 않음
            자료형의 개념이 없는건 아님
            => 변수에 대입되는 값에 따라서 알아서 자료형을 결정함 : 타입추론
  • string : 문자열
  • number : 숫자
  • boolean : 논리
  • object : 객체
  • function : 함수
  • undefined : 초기화가 안된 상태
